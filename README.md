# 🛒 Supermarket Billing System (Python)

A simple **command-line based billing system** built using Python that allows users to select items, calculate total cost, apply GST, and generate a bill.

---

## 🚀 Features

* 📋 Display available items with prices
* 🛍️ Add items with quantity
* 💰 Automatic price calculation
* 🧾 Bill generation with GST (5%)
* 👤 User name and date included in bill
* 🔄 Continuous shopping option

---

## 🧠 Tech Stack

* **Language:** Python
* **Concepts Used:**

  * Dictionaries
  * Lists
  * Loops (while)
  * Conditional statements
  * Functions (basic logic)
  * Date & Time module

---

## ▶️ How It Works

1. User enters their name
2. Menu is displayed:

   * View items
   * Buy items
   * Exit
3. User selects items and quantity
4. Total price is calculated
5. GST (5%) is applied
6. Final bill is generated
